# 🌦️ Atmosphere — Weather App

A beautifully designed, modern weather web application that provides real-time weather updates, air quality data, and forecasts with dynamic visuals and animations.

---

## 🚀 Features

* 🌍 **Search Any City** – Get weather details by entering a city name

* 📍 **Use Current Location** – Automatically detect and display your local weather

* 🌡️ **Current Weather Data**

  * Temperature (°C / °F toggle)
  * Feels like temperature
  * Humidity
  * Wind speed & direction
  * UV Index
  * Cloud cover

* 📊 **Air Quality Index (AQI)**

  * Real-time AQI value
  * Health category & description
  * Pollutants (PM2.5, PM10, CO, NO₂, O₃)

* 🌅 **Sunrise & Sunset Visualization**

  * Animated sun arc based on time of day

* ⏱️ **Hourly Forecast (24 hours)**

* 📅 **7-Day Forecast**

* 🎨 **Dynamic UI Themes**

  * Changes based on weather conditions (rain, snow, clear, etc.)

* 🌧️ **Particle Effects**

  * Rain, snow, and stars animations

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3 (Glassmorphism UI + Animations)**
* **Vanilla JavaScript**
* **APIs Used:**

  * Open-Meteo Weather API
  * Open-Meteo Air Quality API
  * OpenStreetMap (Reverse Geocoding)
